## Usage

```tsx
import {
  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
} from '@tszhong0411/ui'
```

```tsx
<Collapsible>
  <CollapsibleTrigger>Is Next.js a good framework?</CollapsibleTrigger>
  <CollapsibleContent>
    Yes. Next.js is a popular open-source framework for building server-side rendered (SSR) React applications.
  </CollapsibleContent>
</Collapsible>
```
